# The Founders Korea 사이트 기획서 (v3 - 의회 검토 반영)

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | The Founders Korea |
| 성격 | 실전 창업 경영 미디어 플랫폼 + AI/DX 인사이트 + 정부지원사업 DB |
| 운영 주체 | 한국창업경영센터 |
| 기술 스택 | Python (FastAPI), Jinja2, HTML/CSS, Railway 배포 |
| 현재 상태 | MVP - 정적 블로그 (샘플 포스트 3개) |

## 2. 사업 배경

스타트업 투자 유치 위주의 화려한 소식에 가려진 **실전 창업가(Founders)들의 현장 목소리와 경영 실무 정보가 부재**한 상황. 대부분의 창업 미디어는 유니콘 기업, 대규모 투자 소식에 집중하지만, 실제 현장에서 매일 경영 과제를 풀어가는 소상공인과 초기 스타트업 대표들에게 필요한 **실무 중심 정보**는 턱없이 부족합니다.

더 나아가, 정부지원사업 정보는 흩어져 있고 찾기 어려우며, AI/디지털전환에 대한 현장 실무 가이드는 전무한 상황입니다.

## 3. 사업 목적 (4개 축)

1. **실전 경영 + AI/DX 인사이트 제공** - 대한민국 창업가들을 위한 현장 밀착형 경영 정보 + AI/디지털전환 실전 가이드
2. **시민 참여 저널리즘** - 창업가(사장님)가 직접 투고하는 현장 중심 콘텐츠
3. **실용 도구 제공** - 정부지원사업 DB 등 실무에 즉시 활용 가능한 도구 제공
4. **공신력 확보 & 리드 창출** - 한국창업경영센터의 컨설팅 전문성을 미디어화하여 리드 유입

## 4. 핵심 타겟

| 구분 | 대상 | 니즈 |
|------|------|------|
| **Primary** | 현재 사업장을 운영 중인 소상공인 및 초기 스타트업 대표 | 세무, 법률, 경영 실무 정보, 정부지원사업, AI/DX 활용법 |
| **Secondary** | 프랜차이즈 예비 가맹주, 퇴직 후 창업 준비자, 청년 창업자 | 창업 절차, 지원사업 정보, 업종별 실전 가이드 |
| **Tertiary** | 창업 생태계 관계자 (투자자, 컨설턴트, 정책 담당자) | 생태계 동향, 데이터, 정책 효과성 분석 |

### 세부 타겟 페르소나
- 소상공인 (카페, 음식점, 편의점 등)
- 초기 스타트업 (시드~시리즈A)
- 프랜차이즈 예비 가맹주
- 퇴직 후 창업 준비자 (40~50대)
- 청년 창업자 (20~30대)
- 1인 사업자 / 프리랜서

### 초기 집중 페르소나 (v3 신규)

> **초기 6개월은 하이퍼 타겟팅**: 넓은 페르소나를 유지하되, 최초 집중 대상을 명확히 설정.
>
> **1순위 타겟**: B2B SaaS 파운더, 창업 3년 차, 5~10명 규모, 스케일업용 R&D 정부지원사업을 찾는 대표
>
> 이 페르소나를 기준으로 콘텐츠/DB/UX를 설계한 후, 검증된 패턴을 다른 페르소나로 확장.

### Gov-Support Radar 상세 (킬러 피처)
- **508+ 정부지원사업 DB** (중기부, 소상공인진흥공단, 창업진흥원, 지자체 등)
- **검색/필터 기능**: 지역, 업종, 지원유형(자금/교육/컨설팅), 진행상태(모집중/예정/종료)
- **주간 업데이트**: 신규 사업 추가, 마감 임박 알림
- **실시간성**: 공공데이터 API 연동 또는 주간 크롤링
- **사용자 맞춤 알림**: 관심 업종/지역 설정 → 맞춤 알림 발송

#### AI 맞춤 추천 (v3 신규)
- **온보딩 퀴즈**: 업종/사업 단계/규모/지역 입력 → AI 매칭 점수 산출
- **프로그램별 매칭 점수**: 내 사업에 적합도 A/B/C 등급 표시
- **Founder's Pick**: 주간 Top 추천 프로그램 큐레이션

#### 커뮤니티 리뷰 시스템 (v3 신규)
- **성공률 통계**: 프로그램별 실제 선정율, 지원 규모 통계
- **파운더 리뷰**: 실제 신청자/수혜자의 팁, 실패 사례, 주의사항
- **Q&A**: 프로그램별 질문/답변 커뮤니티

#### 신청 키트 (Application Kits) (v3 신규)
- **템플릿**: 주요 프로그램별 사업계획서 템플릿
- **체크리스트**: 필수 서류, 신청 절차 단계별 가이드
- **Step-by-step 가이드**: 인기 프로그램 Top 20 상세 신청 가이드

### 경쟁사 대비 차별화 (v3 신규)

| 항목 | bizinfo.go.kr (정부 포털) | K-Startup | **The Founders Korea** |
|------|--------------------------|-----------|------------------------|
| 데이터 | 원본 데이터 나열 | 정부 공식 포털 | **큐레이션 + AI 매칭** |
| UX | 관료적 UI, 검색 어려움 | 기능 복잡 | **직관적 검색/필터** |
| 맞춤 추천 | 없음 | 기본 필터 | **AI 매칭 점수 + 온보딩 퀴즈** |
| 커뮤니티 | 없음 | 없음 | **파운더 리뷰 + Q&A** |
| 실행 지원 | 없음 | 일부 교육 | **신청 키트 + 컨설팅 연결** |
| 콘텐츠 | 공고문만 | 공고 + 뉴스 | **Actionable Intelligence** (실행 가능한 정보) |

> **핵심 차별화**: bizinfo.go.kr는 "데이터 복제"가 아닌, **"실행 가능한 인텔리전스"**로 차별화.
> 단순 정보 제공 → AI 매칭 + 커뮤니티 리뷰 + 신청 키트 = 원스톱 지원사업 플랫폼

## 6. 비즈니스 모델 (v3 - Full Freemium)

```
완전 Freemium → 데이터 기반 유료화 결정
```

### 수익화 전략 (3단계)

| 단계 | 기간 | 전략 | 상세 |
|------|------|------|------|
| **Phase 1-2** | 0~6개월 | **완전 무료** | 모든 콘텐츠 + DB + 도구 무료 제공. 트래픽 + 신뢰 구축에 집중 |
| **Phase 3** | 6개월+ | **데이터 기반 유료화 결정** | 사용자 행동 데이터 분석 후 유료화 모델 결정 (구독/단품/Value Ladder 중 선택) |
| **상시** | Phase 1~ | **컨설팅 리드** | 컨설팅 상담 연결은 처음부터 상시 활성 (핵심 수익원) |

### Phase 3 유료화 옵션 (데이터 기반 선택)
- **A) 구독 모델**: 프리미엄 DB 기능 + 맞춤 알림 월 구독
- **B) Value Ladder**: 디지털 상품 → 프리미엄 리포트 → 컨설팅 단계적 전환
- **C) 하이브리드**: 구독 + 단품 혼합

> **핵심 원칙**: 6개월간 축적된 사용자 데이터(어떤 콘텐츠에 비용 지불 의향이 있는지)를 기반으로 유료화 결정. 섣부른 유료화로 성장을 저해하지 않음.

### 리드 퍼널 (v3)
```
콘텐츠 열람 → 정부지원사업 DB 활용 → 뉴스레터 구독 → 컨설팅 상담 신청
                                                    (Phase 3 이후 유료 상품 추가 가능)
```
