# The Founders Korea 사이트 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | The Founders Korea |
| 성격 | 실전 창업 경영 미디어 플랫폼 |
| 운영 주체 | 한국창업경영센터 |
| 기술 스택 | Python (FastAPI), Jinja2, HTML/CSS, Railway 배포 |
| 현재 상태 | MVP - 정적 블로그 (샘플 포스트 3개) |

## 2. 사업 배경

스타트업 투자 유치 위주의 화려한 소식에 가려진 **실전 창업가(Founders)들의 현장 목소리와 경영 실무 정보가 부재**한 상황. 대부분의 창업 미디어는 유니콘 기업, 대규모 투자 소식에 집중하지만, 실제 현장에서 매일 경영 과제를 풀어가는 소상공인과 초기 스타트업 대표들에게 필요한 **실무 중심 정보**는 턱없이 부족합니다.

## 3. 사업 목적

1. **실전 경영 인사이트 제공** - 대한민국 창업가들을 위한 현장 밀착형 경영 정보
2. **시민 참여 저널리즘** - 창업가(사장님)가 직접 투고하는 현장 중심 콘텐츠
3. **공신력 확보 & 리드 창출** - 한국창업경영센터의 컨설팅 전문성을 미디어화

## 4. 핵심 타겟

| 구분 | 대상 | 니즈 |
|------|------|------|
| **Primary** | 현재 사업장을 운영 중인 소상공인 및 스타트업 대표 | 세무, 법률, 경영 실무 정보 |
| **Secondary** | 정부지원사업 및 창업을 준비하는 예비 창업자 | 창업 절차, 지원사업 정보 |
| **Tertiary** | 창업 생태계 관계자 (투자자, 컨설턴트, 정책 담당자) | 생태계 동향, 데이터 |

## 5. 핵심 콘텐츠 카테고리 (4개 축)

| 카테고리 | 설명 | 작성자 |
|----------|------|--------|
| **The Insights** | 산업 트렌드, 법률/세무/경영 전문 리포트 | 센터 전문가 |
| **Founder's Voice** | 현장 운영 노하우, 실패 극복기 (시민기자) | 창업가 직접 투고 |
| **Weekly Curation** | 주간 필수 창업 뉴스 요약 (자동화) | 파이썬 스크래퍼 + 편집 |
| **Success Solution** | 컨설팅 성공 케이스 스터디 | 한국창업경영센터 |

## 6. 비즈니스 모델

```
콘텐츠 (무료)  →  신뢰/공신력 축적  →  컨설팅 리드 창출
                                      ↓
                              한국창업경영센터
                              유료 컨설팅 전환
```

- **콘텐츠**: 무료 공개 (트래픽 & 신뢰 확보)
- **리드 창출**: 콘텐츠 → 뉴스레터 구독 → 상담 신청 퍼널
- **브랜딩**: Success Solution 카테고리를 통한 센터 전문성 노출
