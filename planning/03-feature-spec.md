# The Founders Korea 기능 명세서

## Phase 1: 미디어 플랫폼 기반 구축

### F1-1. 5대 카테고리 구조 전환
- **목적**: 단일 블로그 → 5개 콘텐츠 섹션 분리
- **카테고리**:
  - Growth Playbook (`/growth-playbook`) - 창업/경영 실전 가이드
  - AI/DX Lab (`/ai-dx-lab`) - AI 도구 리뷰 & 디지털 전환 사례
  - Founder's Voice (`/founders-voice`) - 시민기자 투고 콘텐츠
  - Gov-Support Radar (`/gov-support`) - 정부지원사업 DB & 주간 공고
  - Success Cases (`/success-cases`) - 한국창업경영센터 케이스 스터디
- **구현**: 카테고리별 목록 페이지 + 상세 페이지
- **우선순위**: 높음

### F1-2. 데이터베이스 도입
- **목적**: 하드코딩 데이터를 DB로 이관
- **기술**: SQLite + SQLAlchemy (초기), PostgreSQL (운영)
- **모델**:
  - `Post`: id, title, slug, summary, content, category, author_name, tags, is_published, created_at, updated_at
  - `GovSupportProgram`: id, 사업명, 주관기관, 신청기간_시작, 신청기간_종료, 지원대상, 지원내용, 신청방법, 원문링크, 태그, 상태(모집중/마감/예정), 지역, 업종, 지원유형, created_at, updated_at
  - `Tag`: id, name, slug
  - `Category`: id, name, slug, description (5개 고정)
- **우선순위**: 높음

### F1-3. 정부지원사업 DB MVP
- **목적**: Gov-Support Radar 핵심 기능 - 실시간 정부지원사업 검색 DB
- **기능**:
  - **검색 필터**: 지역, 업종, 지원유형(R&D/자금/공간/판로 등), 상태(모집중/예정/마감)
  - **목록 페이지**: 카드형 리스트, 마감일 D-day 표시, 상태 뱃지
  - **상세보기**: 사업명, 주관기관, 신청기간, 지원대상, 지원내용, 신청방법, 원문 링크
  - **데이터 수집**:
    - 공공데이터포털 API (정부지원사업 통합공고)
    - K-Startup 크롤링
    - 중소벤처기업부 웹사이트 스크래핑
  - **업데이트**: 주 2회 자동 수집 + 수동 검수
- **기술**: `requests` + `BeautifulSoup` / API 연동
- **우선순위**: 높음

### F1-3a. AI 맞춤 추천 (온보딩 퀴즈)
- **목적**: 정부지원 DB 차별화 - 사용자 프로필 기반 AI 매칭 (vs bizinfo.go.kr 단순 검색과의 차별점)
- **기능**:
  - 온보딩 퀴즈 (업종, 창업단계, 규모, 지역, 관심분야)
  - AI 매칭 점수 표시 (각 프로그램에 "내 사업 적합도 85%" 등)
  - 사용자 프로필 저장 → 재방문 시 자동 추천
- **우선순위**: 높음

### F1-3b. 커뮤니티 리뷰 시스템
- **목적**: 각 정부지원 프로그램에 창업자 리뷰/팁 추가 (bizinfo.go.kr에 없는 실사용자 경험)
- **기능**:
  - 별점 (1~5), 텍스트 리뷰, 성공/실패/진행중 태그, 신청 팁
  - 성공률 통계 표시 (리뷰 데이터 기반)
  - 프로그램 상세 페이지 내 리뷰 섹션
- **우선순위**: 높음

### F1-3c. 신청 키트 (Application Kit)
- **목적**: 인기 프로그램에 대한 실행 가이드 제공
- **기능**:
  - 다운로드 가능한 체크리스트, 서류 준비 가이드, 사업계획서 팁
  - Phase 1에서는 상위 20% 인기 프로그램만 제공
  - 마크다운 기반 콘텐츠 + JSON 체크리스트
- **우선순위**: 중간

### F1-4. 마크다운 렌더링
- **목적**: 전문 리포트 및 투고 글의 풍부한 본문 표현
- **기술**: `python-markdown` + 확장 (코드 하이라이팅, 테이블, TOC)
- **우선순위**: 높음

### F1-5. SEO 최적화
- **구현 항목**:
  - Open Graph / Twitter Card 메타 태그
  - `sitemap.xml` 자동 생성
  - `robots.txt`
  - JSON-LD 구조화 데이터 (Article, Organization, GovernmentService)
  - 카테고리별 canonical URL
- **우선순위**: 높음

### F1-6. 메인 페이지 개편
- **구현**: 히어로 + 5대 카테고리별 최신 콘텐츠 미리보기 레이아웃
- **특별 섹션**:
  - Gov-Support Radar: "마감 임박 정부지원사업 Top 5" 위젯
  - Success Cases: CTA 버튼 강화 ("무료 상담 신청")
- **우선순위**: 높음

### F1-7. RSS 피드
- **경로**: `/feed.xml` (전체), 카테고리별 RSS 별도 제공
- **추가**: `/gov-support/feed.xml` (정부지원사업 신규 공고 전용)
- **우선순위**: 중간

### F1-8. 페이지네이션
- **구현**: 카테고리별 목록 페이지 10개 단위 페이징
- **우선순위**: 중간

## Phase 2: CMS & 수익화 기반

### F2-1. 관리자 인증 & 대시보드
- **기능**: 로그인, 포스트 CRUD, 카테고리별 관리, 정부지원사업 DB 관리
- **인증**: 비밀번호 기반 (초기)
- **우선순위**: 높음

### F2-2. 마크다운 에디터
- **기능**: 실시간 미리보기, 카테고리 선택, 태그 지정, 발행/임시저장
- **우선순위**: 높음

### F2-3. Founder's Voice 투고 시스템
- **목적**: 시민기자(창업가)가 직접 글을 투고
- **기능**:
  - 공개 투고 폼 (`/founders-voice/submit`)
  - 작성자 정보 (이름, 사업장, 간단 소개)
  - 마크다운 또는 일반 텍스트 본문
  - 관리자 검토 → 승인 후 발행
  - 투고 가이드라인 안내
- **우선순위**: 높음

### F2-4. Gov-Support Radar 스크래퍼 & 큐레이션
- **목적**: 정부지원사업 자동 수집 + 주간 뉴스 큐레이션
- **기능**:
  - **자동 스크래퍼**: 정부지원사업 DB 자동 업데이트 (주 2회)
  - **주간 뉴스**: 창업/정책 관련 주요 뉴스 수집
  - 뉴스 소스: 정부 공고, 주요 경제지, 스타트업 미디어
  - 수집된 데이터 관리자 검토 → 편집 → 발행
- **기술**: `requests` + `BeautifulSoup` / `Scrapy`
- **우선순위**: 높음

### F2-5. 디지털 상품 시스템 → **Phase 3으로 이동 (Freemium 전략)**
- ~~**목적**: 수익화 1단계 - 저가 디지털 상품 판매~~
- **변경 사유**: Freemium 전략에 따라 Phase 1-2에서는 모든 콘텐츠 무료 제공. 수익화는 Phase 3에서 데이터 기반 결정
- **원래 계획**: 사업계획서 템플릿, 체크리스트, 재무 계산기 등 → Phase 3 수익화 시점에 재검토
- **우선순위**: ~~중간~~ → Phase 3 이동

> **참고**: Phase 2에서는 수익화보다 **사용자 확보와 데이터 수집**에 집중. 컨설팅 리드(상담 신청)는 Phase 1부터 활성화.

### F2-6. 정부지원 맞춤 알림
- **목적**: 구독자 개인화 서비스
- **기능**:
  - 구독자가 관심 분야 등록 (지역, 업종, 지원유형)
  - 새로운 정부지원사업 공고 → 조건 매칭 → 이메일/카카오톡 알림
  - 알림 설정 관리 (빈도, 채널)
- **기술**: 백그라운드 잡 (Celery / APScheduler) + 이메일/알림톡 API
- **우선순위**: 중간

### F2-7. 검색 기능
- **구현**: 제목/본문 전문 검색, 카테고리 필터 결합
- **추가**: 정부지원사업 DB 검색 통합
- **우선순위**: 중간

### F2-8. 태그 시스템
- **기능**: 다중 태그, 태그별 포스트 목록, 카테고리 횡단 태그
- **우선순위**: 중간

### F2-9. 뉴스레터 구독
- **목적**: 이메일 기반 구독자 확보 (리드 퍼널 1단계)
- **기능**:
  - 이메일 수집 폼 (메인 + 전용 페이지 + 포스트 하단)
  - 구독 확인, 해지
  - 주간 뉴스레터 발송 (Gov-Support Radar 주간 요약 + 인기 콘텐츠)
- **우선순위**: 중간

### F2-10. 소셜 공유
- **기능**: 카카오톡, X, LinkedIn, 링크 복사
- **우선순위**: 낮음

## Phase 3: 프리미엄 서비스 & 확장

### F3-1. 상담 신청 폼 (`/contact`)
- **목적**: 한국창업경영센터 컨설팅 리드 확보
- **기능**:
  - 상담 분야 선택 (법률, 세무, 경영, 정부지원사업 등)
  - 사업자 정보 + 문의 내용
  - 관리자 이메일 알림
  - CRM 연동 (선택)
- **우선순위**: 높음

### F3-2. Success Cases CTA 강화
- **목적**: 케이스 스터디 → 상담 전환율 극대화
- **기능**:
  - 각 Success Cases 글 하단에 "유사 사례 무료 상담받기" CTA
  - 상담 신청 폼 프리필 (해당 케이스 분야 자동 선택)
  - 성공 사례 페이지 전용 랜딩 요소 (Before/After, ROI 수치 강조)
- **우선순위**: 높음

### F3-3. AI 도구 제휴/어필리에이트
- **목적**: AI/DX Lab 콘텐츠 수익화
- **기능**:
  - AI 도구 리뷰 글에 제휴 링크 삽입
  - 제휴 프로그램: ChatGPT, Notion AI, Jasper, Copy.ai 등
  - 클릭/전환 추적
- **예상 수익**: 건당 수수료 또는 월 구독 커미션
- **우선순위**: 중간

### F3-4. 프리미엄 리포트/교육 콘텐츠
- **목적**: 고가 디지털 상품 판매
- **상품 예시**:
  - 업종별 심화 리포트 (50~100p)
  - 온라인 강의 (녹화형)
  - 웨비나 티켓 (라이브)
- **가격대**: 50,000원 ~ 300,000원
- **우선순위**: 중간

### F3-5. 프리미엄 구독 서비스 (Freemium 데이터 기반 결정)
- **목적**: 정기 수익 모델 (MRR) - **6개월 사용자 데이터 분석 후 최적 모델 선택**
- **수익화 결정 시점**: Phase 1-2 동안 수집된 사용자 행동 데이터 분석 후 결정
  - 어떤 콘텐츠/기능에 가장 높은 engagement가 있는지
  - 사용자가 가장 가치를 느끼는 기능이 무엇인지
  - 유료 전환 의향 서베이 결과 반영
- **후보 혜택** (데이터 기반 확정):
  - 전체 프리미엄 콘텐츠 무제한 액세스
  - 정부지원 맞춤 알림 우선 제공
  - 월 1회 온라인 Q&A 세션 참여
  - 프리미엄 커뮤니티 (디스코드/슬랙)
  - 디지털 상품 할인
- **가격**: 데이터 기반 결정 (기존 안: 월 29,000원 / 연 290,000원)
- **우선순위**: 낮음 (Phase 3 후반)

### F3-6. 댓글 시스템
- **옵션**: Giscus (GitHub 기반) 또는 자체 구현
- **우선순위**: 낮음

### F3-7. 시민기자 프로필
- **기능**: 투고 이력, 사업장 소개, 작성 글 모아보기
- **우선순위**: 낮음

### F3-8. 분석 & 리포팅
- **기능**: 방문자 통계, 인기 콘텐츠, 구독 전환율, 상품 판매 분석
- **기술**: 자체 또는 Google Analytics / Plausible
- **우선순위**: 낮음
