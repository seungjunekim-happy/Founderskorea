# The Founders Korea 기능 명세서

## Phase 1: 미디어 플랫폼 기반 구축

### F1-1. 4대 카테고리 구조 전환
- **목적**: 단일 블로그 → 4개 콘텐츠 섹션 분리
- **카테고리**:
  - The Insights (`/insights`)
  - Founder's Voice (`/founders-voice`)
  - Weekly Curation (`/weekly`)
  - Success Solution (`/success-solution`)
- **구현**: 카테고리별 목록 페이지 + 상세 페이지
- **우선순위**: 높음

### F1-2. 데이터베이스 도입
- **목적**: 하드코딩 데이터를 DB로 이관
- **기술**: SQLite + SQLAlchemy (초기), PostgreSQL (운영)
- **모델**:
  - `Post`: id, title, slug, summary, content, category, author_name, tags, is_published, created_at, updated_at
  - `Tag`: id, name, slug
  - `Category`: id, name, slug, description (4개 고정)
- **우선순위**: 높음

### F1-3. 마크다운 렌더링
- **목적**: 전문 리포트 및 투고 글의 풍부한 본문 표현
- **기술**: `python-markdown` + 확장 (코드 하이라이팅, 테이블, TOC)
- **우선순위**: 높음

### F1-4. SEO 최적화
- **구현 항목**:
  - Open Graph / Twitter Card 메타 태그
  - `sitemap.xml` 자동 생성
  - `robots.txt`
  - JSON-LD 구조화 데이터 (Article, Organization)
  - 카테고리별 canonical URL
- **우선순위**: 높음

### F1-5. 메인 페이지 개편
- **구현**: 히어로 + 카테고리별 최신 콘텐츠 미리보기 레이아웃
- **우선순위**: 높음

### F1-6. RSS 피드
- **경로**: `/feed.xml` (전체), 카테고리별 RSS 별도 제공 가능
- **우선순위**: 중간

### F1-7. 페이지네이션
- **구현**: 카테고리별 목록 페이지 10개 단위 페이징
- **우선순위**: 중간

## Phase 2: CMS & 시민기자 시스템

### F2-1. 관리자 인증 & 대시보드
- **기능**: 로그인, 포스트 CRUD, 카테고리별 관리
- **인증**: 비밀번호 기반 (초기)
- **우선순위**: 높음

### F2-2. 마크다운 에디터
- **기능**: 실시간 미리보기, 카테고리 선택, 태그 지정, 발행/임시저장
- **우선순위**: 높음

### F2-3. Founder's Voice 투고 시스템
- **목적**: 시민기자(창업가)가 직접 글을 투고
- **기능**:
  - 공개 투고 폼 (`/founders-voice/submit`)
  - 작성자 정보 (이름, 사업장, 간단 소개)
  - 마크다운 또는 일반 텍스트 본문
  - 관리자 검토 → 승인 후 발행
  - 투고 가이드라인 안내
- **우선순위**: 높음

### F2-4. Weekly Curation 자동화
- **목적**: 주간 창업 뉴스 자동 수집 & 요약
- **기능**:
  - 파이썬 스크래퍼로 주요 창업/경영 뉴스 수집
  - 뉴스 소스: 정부 공고, 주요 경제지, 스타트업 미디어
  - 수집된 뉴스 관리자 검토 → 편집 → 발행
  - 매주 자동 또는 반자동 발행
- **기술**: `requests` + `BeautifulSoup` 또는 `Scrapy`
- **우선순위**: 중간

### F2-5. 검색 기능
- **구현**: 제목/본문 전문 검색, 카테고리 필터 결합
- **우선순위**: 중간

### F2-6. 태그 시스템
- **기능**: 다중 태그, 태그별 포스트 목록, 카테고리 횡단 태그
- **우선순위**: 중간

### F2-7. 뉴스레터 구독
- **목적**: 이메일 기반 구독자 확보 (리드 퍼널 1단계)
- **기능**:
  - 이메일 수집 폼 (메인 + 전용 페이지 + 포스트 하단)
  - 구독 확인, 해지
  - 주간 뉴스레터 발송 (Weekly Curation 연동)
- **우선순위**: 중간

### F2-8. 소셜 공유
- **기능**: 카카오톡, X, LinkedIn, 링크 복사
- **우선순위**: 낮음

## Phase 3: 리드 퍼널 & 확장

### F3-1. 상담 신청 폼 (`/contact`)
- **목적**: 한국창업경영센터 컨설팅 리드 확보
- **기능**:
  - 상담 분야 선택 (법률, 세무, 경영, 정부지원사업 등)
  - 사업자 정보 + 문의 내용
  - 관리자 이메일 알림
  - CRM 연동 (선택)
- **우선순위**: 높음

### F3-2. Success Solution CTA 강화
- **목적**: 케이스 스터디 → 상담 전환
- **기능**: 각 Success Solution 글 하단에 "유사 사례 상담받기" CTA
- **우선순위**: 중간

### F3-3. 댓글 시스템
- **옵션**: Giscus (GitHub 기반) 또는 자체 구현
- **우선순위**: 낮음

### F3-4. 시민기자 프로필
- **기능**: 투고 이력, 사업장 소개, 작성 글 모아보기
- **우선순위**: 낮음

### F3-5. 분석 & 리포팅
- **기능**: 방문자 통계, 인기 콘텐츠, 구독 전환율
- **기술**: 자체 또는 Google Analytics / Plausible
- **우선순위**: 낮음
