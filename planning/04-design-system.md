### 카테고리별 컬러 (5개)
1. **Growth Playbook** (성장 플레이북)
   - Color: #3b82f6 (블루)

   - 용도: 시장 분석, 인사이트, 트렌드 리포트

2. **AI/DX Lab** (NEW)
   - Color: #06b6d4 (시안/틸)
   - 의미: 기술, 혁신, 디지털 전환
   - 용도: AI 활용, 디지털 도구, 자동화 솔루션

3. **Founder's Voice** (창업자의 목소리)
   - Color: #f59e0b (앰버)
   - 의미: 현장감, 따뜻함, 공감
   - 용도: 창업 경험담, 실전 노하우, 커뮤니티

4. **Gov-Support Radar** (정부지원 레이더)
   - Color: #10b981 (에메랄드)
   - 의미: 실용성, 정부 지원, 기회 포착
   - 용도: 지원사업 정보, 정책 소식, 실시간 업데이트

5. **Success Cases** (성공 사례)
   - Color: #8b5cf6 (바이올렛)
   - 의미: 성과, 전문 컨설팅, 검증된 솔루션
   - 용도: 컨설팅 후기, 성공 스토리, 디지털 상품

## 5. 반응형 디자인

### 브레이크포인트
- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

### 모바일 최적화
- 카드 그리드: 1열
- 폰트 크기: 90%
- 패딩: 1rem
- 네비게이션: 햄버거 메뉴

### 태블릿
- 카드 그리드: 2열
- 폰트 크기: 95%
- 패딩: 2rem

### 데스크톱
- 카드 그리드: 3열
- 폰트 크기: 100%
- 패딩: 4rem

## 6. 애니메이션

### 전환 효과
```css
transition: all 0.3s ease-in-out
```

### 호버 효과
- 카드: `transform: translateY(-4px)` + `box-shadow` 증가
- 버튼: 배경색 변경
- 링크: 밑줄 애니메이션

### 로딩
- 스켈레톤 UI: 회색 그라데이션 애니메이션
- 스피너: 브랜드 컬러 회전 애니메이션

## 9. 접근성 (Accessibility)

- **색상 대비**: WCAG AA 기준 준수 (4.5:1)
- **키보드 네비게이션**: 모든 인터랙티브 요소 접근 가능
- **스크린 리더**: aria-label, alt 텍스트 제공
- **포커스 표시**: 명확한 포커스 아웃라인

## 10. 성능 최적화

- **이미지**: WebP 포맷, lazy loading
- **폰트**: 가변 폰트, font-display: swap
- **CSS**: Critical CSS 인라인
- **JS**: 코드 스플리팅, 트리 쉐이킹
