# The Founders Korea 디자인 시스템

## 1. 디자인 컨셉

**"신뢰감 있는 미디어 + 현장의 진정성"**

- 전문 미디어로서의 공신력과 깔끔함
- 창업 현장의 생생함을 담는 따뜻한 톤
- 콘텐츠 가독성 최우선
- 참고 레퍼런스: The Information, Substack, 폴인(folin), 아웃스탠딩

## 2. 컬러 팔레트

```
배경        --bg:              #0a0a0a

테두리      --border:          #262626
텍스트      --text:            #fafafa

액센트 호버 --accent-hover:    #60a5fa
태그 배경   --tag-bg:          #1e3a5f

| 카테고리 | 주 색상 | 용도 |
|----------|--------|------|
| The Insights | `#3b82f6` (블루) | 전문성, 분석 |
| Founder's Voice | `#f59e0b` (앰버) | 현장감, 따뜻함 |

```
카테고리 컬러 적용 위치:
  - 카테고리 배지 (태그 스타일)

  - 포스트 카드 좌측 보더 (2px 컬러 라인)
```

## 3. 타이포그래피

| 용도 | 크기 | 두께 | 비고 |
|------|------|------|------|
| 사이트 제목 (히어로) | 2.5rem | 800 | 그래디언트 또는 단색 |
| 카테고리 헤딩 | 1.8rem | 700 | 카테고리 목록 상단 |
| 포스트 제목 (상세) | 2rem | 800 | |
| 카드 제목 | 1.2rem | 700 | |
| 본문 | 기본 | 일반 | line-height: 1.8 (가독성 강화) |
| 메타 정보 | 0.8~0.85rem | - | 보조 색상 |
| 카테고리 배지 | 0.75rem | 600 | 카테고리별 컬러 |

### 한글 폰트
- **제안**: Pretendard 또는 Noto Sans KR 웹폰트
- **이유**: 본문 가독성이 핵심인 미디어 사이트에서 시스템 폰트 대비 가독성 향상

## 4. 컴포넌트

### 헤더

```
┌───────────────────────────────────────────────────────────────┐
│  THE FOUNDERS KOREA    Insights  Voice  Weekly  Solution  소개 │
└───────────────────────────────────────────────────────────────┘
```

- 4대 카테고리가 주 네비게이션
- sticky + 배경 블러
- 모바일: 햄버거 메뉴

### 포스트 카드

```

## 5. 레이아웃

- **콘텐츠 최대 너비**: 720px (본문), 960px (목록)
- **메인 페이지**: 단일 컬럼, 카테고리별 섹션 배치
- **카테고리 목록**: 카드 리스트
- **포스트 상세**: 단일 컬럼, 본문 중심
- **반응형 기준**: 768px (태블릿), 640px (모바일)

## 6. 향후 디자인 개선

1. **라이트/다크 모드 토글** - 미디어 사이트는 라이트 모드 수요도 높음
2. **커버 이미지** - 포스트별 대표 이미지 (OG 이미지 겸용)
3. **코드 하이라이팅** - 기술 관련 Insights 포스트용
4. **인포그래픽** - Weekly Curation 데이터 시각화
5. **인쇄용 스타일** - 리포트 형태 콘텐츠의 인쇄/PDF 지원
