# The Founders Korea 콘텐츠 전략

## 1. 4대 콘텐츠 카테고리 상세

### The Insights (전문 리포트)

| 항목 | 내용 |
|------|------|
| 작성자 | 한국창업경영센터 전문가 |
| 주제 | 산업 트렌드, 법률/세무/경영 분석 리포트 |
| 빈도 | 주 1~2회 |
| 톤 | 전문적, 분석적, 데이터 기반 |
| 길이 | 2,000~5,000자 (읽기 10~20분) |
| 예시 콘텐츠 |
| - 2026년 소상공인 세무 변경사항 총정리 |
| - 프랜차이즈 창업 시 반드시 확인할 법률 5가지 |
| - 정부지원사업 합격률 높이는 사업계획서 작성법 |
| - AI 도입으로 인건비 30% 절감한 소상공인 사례 분석 |

### Founder's Voice (시민기자 투고)

| 항목 | 내용 |
|------|------|
| 작성자 | 현직 창업가 (사장님) 직접 투고 |
| 주제 | 현장 운영 노하우, 실패 극복기, 일상 |
| 빈도 | 주 2~3회 (투고량에 따라) |
| 톤 | 솔직한, 현장감 있는, 1인칭 서사 |
| 길이 | 1,000~3,000자 (읽기 5~15분) |
| 검수 | 관리자 검토 → 최소 편집 후 발행 |
| 예시 콘텐츠 |
| - 배달앱 수수료와 싸우며 배운 것들 (치킨집 사장님) |
| - 직원 3명에서 30명까지, 성장통 극복기 |
| - 망한 카페를 살린 한 가지 결정 |
| - 정부지원사업, 실제로 받아보니 (현실 후기) |

### Weekly Curation (주간 뉴스 요약)

| 항목 | 내용 |
|------|------|
| 작성 | 파이썬 스크래퍼 수집 + 편집자 선별/요약 |
| 주제 | 주간 필수 창업/경영 뉴스 큐레이션 |
| 빈도 | 매주 월요일 발행 |
| 형식 | 뉴스 5~10건 요약 + 편집자 코멘트 |
| 소스 | 정부 공고, 경제지, 스타트업 미디어, 정책 브리핑 |
| 예시 콘텐츠 |
| - [2026년 7주차] 이번 주 창업가가 알아야 할 뉴스 |
|   - 중소벤처기업부 신규 지원사업 3건 공고 |
|   - 최저임금 인상안 확정, 소상공인 영향은? |
|   - AI 규제 법안 국회 통과, 스타트업 대응 방안 |

### Success Solution (케이스 스터디)

| 항목 | 내용 |
|------|------|
| 작성자 | 한국창업경영센터 |
| 주제 | 실제 컨설팅 성공 사례 (익명화) |
| 빈도 | 월 2~4회 |
| 톤 | 스토리텔링 + 전문 분석 |
| 길이 | 2,000~4,000자 |
| 목적 | 센터 전문성 노출 → 상담 리드 창출 |
| 예시 콘텐츠 |
| - 매출 3억 정체 음식점, 법인 전환 후 달라진 세금 |
| - 정부지원사업 3회 탈락 후 합격한 사업계획서의 비밀 |
| - 공동창업자 분쟁, 주주간 계약서로 해결한 사례 |

## 2. 콘텐츠 톤 & 가이드라인

### 전체 톤
- **전문적이되 쉬운** - 전문 용어는 반드시 풀어서 설명
- **현장 중심** - 이론보다 실제 사례와 경험
- **실행 가능한** - 읽고 바로 적용할 수 있는 정보
- **존댓말** - 친근하면서도 신뢰감 있는 존댓말체

### Founder's Voice 투고 가이드라인
1. 실제 경험에 기반한 내용일 것
2. 특정 업체 비방이나 광고성 내용 불가
3. 사진/자료 첨부 권장
4. 최소 800자 이상
5. 편집팀이 최소한의 교정 후 발행

## 3. SEO 키워드 전략

### 카테고리별 주요 키워드

| 카테고리 | 키워드 |
|----------|--------|
| Insights | 소상공인 세금, 창업 법률, 경영 전략, 정부지원사업 |
| Voice | 창업 후기, 사장님 이야기, 자영업 현실, 폐업 극복 |
| Weekly | 창업 뉴스, 소상공인 정책, 스타트업 소식 |
| Solution | 경영 컨설팅 사례, 사업계획서, 법인 전환 |

### 롱테일 키워드 예시
- 2026년 소상공인 세금 절약 방법
- 정부지원사업 사업계획서 쓰는 법
- 프랜차이즈 창업 비용 현실
- 1인 법인 설립 장단점
- 소상공인 정책자금 신청 방법

## 4. 콘텐츠 → 리드 전환 전략

```
콘텐츠 소비  →  뉴스레터 구독  →  상담 신청  →  컨설팅 계약
  (무료)        (이메일 확보)     (리드 확보)    (매출)
```

### 전환 포인트
1. **포스트 하단 CTA** - "이 주제에 대해 더 알고 싶으시면 상담을 신청하세요"
2. **Success Solution 하단** - "비슷한 고민이 있으신가요? 무료 상담 받기"
3. **뉴스레터** - 매주 뉴스레터에 상담 신청 링크 포함
4. **메인 페이지** - 상단/하단에 상담 CTA 배치
