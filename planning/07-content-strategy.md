## 2. 콘텐츠 톤 & 가이드라인

### 전체 톤
- **전문적이되 쉬운** - 전문 용어는 반드시 풀어서 설명
- **현장 중심** - 이론보다 실제 사례와 경험
- **실행 가능한** - 읽고 바로 적용할 수 있는 정보
- **존댓말** - 친근하면서도 신뢰감 있는 존댓말체

### Founder's Voice 투고 가이드라인
1. 실제 경험에 기반한 내용일 것
2. 특정 업체 비방이나 광고성 내용 불가
3. 사진/자료 첨부 권장
4. 최소 800자 이상
5. 편집팀이 최소한의 교정 후 발행

### AI/DX Lab 콘텐츠 가이드라인
1. 실제 도구/서비스를 직접 테스트 후 작성
2. 비용, 장단점, 도입 난이도 명시
3. 스크린샷, 튜토리얼 이미지 필수
4. 소상공인/중소기업 관점에서 평가
5. 광고/협찬 여부 명확히 표기

## 4. 콘텐츠 → 리드 전환 전략 (Freemium 모델)

> **Phase 1-2: 모든 콘텐츠 무료. 유료화는 Phase 3에서 데이터 기반 결정.**

```
콘텐츠 소비 → 온보딩 퀴즈 → 뉴스레터/알림 구독 → 상담 신청 → 컨설팅 계약
  (무료)      (프로필 확보)   (이메일 확보)        (리드 확보)    (매출)
```

### 전환 포인트

#### 1. 포스트 하단 CTA
- "이 주제에 대해 더 알고 싶으시면 상담을 신청하세요"
- "비슷한 고민이 있으신가요? 무료 상담 받기"

#### 2. Success Cases 하단 CTA
- "우리 회사도 이런 결과를 만들고 싶다면? 무료 진단 받기"
- "전문가와 30분 무료 상담 신청하기"

#### 3. 온보딩 퀴즈 → AI 맞춤 추천 전환 경로 (NEW)
```
콘텐츠 소비 → 온보딩 퀴즈 (5개 질문) → AI 맞춤 추천 결과
                                              ↓
                                   맞춤 알림 구독 (이메일 확보)
                                              ↓
                                   "신청 도움받기" CTA → 상담 신청
```
- 메인 페이지, Gov-Support DB 등에서 "내 사업 맞춤 추천받기" CTA로 퀴즈 유도
- 퀴즈 완료 시 AI가 적합도 기반 정부지원사업 매칭
- 매칭 결과 페이지에서 이메일 알림 구독 + 상담 신청 전환

#### 4. Gov-Support Radar DB 전환 경로
```
DB 검색 → 지원사업 상세페이지 → "이 사업 신청 도움받기" CTA
                                      ↓
                                  상담 신청 폼
                                  (지원사업명 자동 입력)
```
- 예: "소상공인 디지털 전환 지원사업" 페이지 하단에 "신청서 작성 도움 받기" 버튼
- 클릭 시 상담 신청 폼 (지원사업 정보 자동 연결)
- 상세 페이지에 커뮤니티 리뷰 + 신청 키트(무료) 제공으로 체류 시간 증가

#### 5. 뉴스레터
- 매주 뉴스레터에 상담 신청 링크 포함
- 지원사업 마감 알림 메일에 "신청 도움 받기" CTA

#### 6. 메인 페이지
- 상단/하단에 상담 CTA 배치
- "내 사업 맞춤 지원사업 AI 추천받기" CTA 배치
- 팝업: 첫 방문자 대상 "무료 경영 진단 받기"

## 6. GTM (Go-to-Market) 전략

### SEO 전략
- **롱테일 키워드 집중**: "정부지원사업 + 업종별" 조합으로 검색 유입 확보
  - 예: "음식점 정부지원사업", "IT 스타트업 지원금", "제조업 R&D 지원"
- 카테고리별 SEO 최적화된 랜딩 페이지 제작
- 정부지원사업 DB 페이지 자체가 SEO 자산 (508개+ 개별 페이지)

### 파트너십
- **액셀러레이터**: 스타트업 액셀러레이터 프로그램과 콘텐츠 제휴
- **코워킹스페이스**: WeWork, 패스트파이브, 마루180 등 입주 창업자 대상 콘텐츠 배포
- **대학 창업센터**: 예비 창업자 대상 정부지원사업 가이드 공동 제작

### 커뮤니티 참여
- **디스콰이엇**: 프로덕트 런칭, 빌더 커뮤니티에서 활발한 참여 (스팸이 아닌 실질적 도움 제공)
- **네이버 카페**: 자영업자/소상공인 커뮤니티에서 정부지원사업 정보 공유 및 질문 답변
- **핵심 원칙**: 홍보가 아닌 도움. 커뮤니티에 가치를 먼저 제공

### 뉴스레터 = 1차 제품
- 플랫폼 완성 전에도 뉴스레터부터 시작 가능
- 매주 월요일 발행: 주간 정부지원사업 + 창업 인사이트 큐레이션
- 뉴스레터 구독자 = 초기 사용자 풀, 플랫폼 런칭 시 첫 트래픽 확보

### 리드 마그넷
- **"내 업종별 정부지원사업 TOP 10" 체크리스트**: 이메일 입력 시 PDF 다운로드
- 업종별 (음식점, IT, 제조업, 유통 등) 각각 제작하여 타겟팅
- 온보딩 퀴즈 자체가 리드 마그넷 역할 (퀴즈 완료 → 결과 이메일 발송)

### 카카오 채널
- 카카오톡 채널 개설하여 정부지원사업 마감 알림, 새 콘텐츠 알림 발송
- 1:1 채팅으로 간단한 문의 응대
- 카카오 채널 친구 추가 → 뉴스레터 구독 전환 유도

## 7. One Source Multi-Use 원칙

### 콘텐츠 재활용 파이프라인

```
블로그 포스트 (원본)
    ↓
뉴스레터 (요약 + 링크)
    ↓
소셜 미디어 카드 (인포그래픽)
    ↓
쇼트폼 영상 (60초 요약)
    ↓
팟캐스트/오디오 (낭독 버전)
```

### 카테고리별 재활용 전략

### 숏폼 영상 전략 (Phase 2 필수)

> 2026년 트렌드: 숏폼 영상이 콘텐츠 소비의 핵심 채널. Phase 2부터 반드시 시작.

- **정부지원 DB 마감 알림 → 30초 숏폼** (인스타 릴스, 유튜브 쇼츠)
  - "이번 주 마감! 소상공인 디지털전환 지원사업 300만원" 형태
  - 매주 2~3개, 템플릿 기반 빠른 제작
- **Founder's Voice → 60초 인터뷰 클립**
  - 사장님 인터뷰 핵심 한마디를 숏폼으로 제작
  - "3년차 카페 사장님이 말하는 정부지원사업 꿀팁" 형태
- **AI/DX Lab → 30초 Before/After 시연**
  - AI 도구 도입 전후 비교 화면 녹화
- **채널**: 인스타그램 릴스, 유튜브 쇼츠, 틱톡 (우선순위 순)

1. **블로그 발행** (매주 5~7개 포스트)
