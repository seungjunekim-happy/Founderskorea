# The Founders Korea UI 화면 설계서

> 사용자가 실제로 보게 되는 모든 화면을 정의합니다.
> 각 화면의 레이아웃, 구성 요소, 인터랙션을 와이어프레임으로 기술합니다.

---

## 공통 요소

### 헤더 (모든 페이지 상단 고정)

```
┌────────────────────────────────────────────────────────────────────────┐
│  THE FOUNDERS KOREA    Playbook  AI/DX  Voice  Gov-Support  Cases  소개│
└────────────────────────────────────────────────────────────────────────┘
  ↑ 로고 (홈 이동)        ↑ 5대 카테고리 메뉴                     ↑ 부가 메뉴

  · 상단 고정(sticky), 배경 블러
  · 현재 카테고리 메뉴 하이라이트 (해당 카테고리 색상 밑줄)
  · 모바일(768px 이하): 햄버거 메뉴로 전환
```
