# The Founders Korea 사업 전략 (v4 - 삼각 플랫폼 전환)

> **v4 주요 변경 사항**
>
> | 항목 | v3 | v4 |
> |------|----|----|
> | 플랫폼 구조 | 4대 축 (창업자 중심 단방향) | **삼각 플랫폼** (창업자 ↔ 투자자 ↔ 정부) |
> | 타겟 고객 | 1차: 소상공인/창업자 | **2대 고객군**: 창업자 + 투자자 |

> | 수익 모델 | B2C 중심 | **B2C + B2B(투자자) + B2G** 삼각 수익 |

## 1. 사업 배경

### 1.1 시장 문제 인식

한국 창업 생태계는 **정보의 삼중 단절(Triple Information Gap)** 상태에 놓여 있다.

```
┌─────────────────┐
│    정부 (Gov)     │  ~9조원 창업지원 예산
│ "성과가 안 보인다" │  집행률 높으나 추적 불가
└──────┬──────────┘
       │ ← 정보 단절 ①
       ▼
┌─────────────────┐
│ 창업자 (Founders) │  연 15만+ 신규 창업
│ "정보를 못 찾겠다" │  지원사업 탈락률 85%+
└──────┬──────────┘
       │ ← 정보 단절 ②
       ▼
┌─────────────────┐
│ 투자자 (Investors)│  초기 딜소싱 사각지대
│ "좋은 딜을 못 찾겠다"│  정부지원 이력 검증 불가
└─────────────────┘
       ▲
       │ ← 정보 단절 ③ (정부↔투자자 직접 연결 부재)
```

### 1.2 창업자 측 Pain Points

| 문제 | 현황 (2026) | 데이터 소스 |
|------|-------------|-------------|
| 정보 분산 | 정부지원사업 **연 3,000개+** 분산 공고 | 중소벤처기업부 |
| 낮은 접근성 | 소상공인 디지털 정보 접근률 **34.2%** | 소상공인시장진흥공단 |
| 높은 실패율 | 5년 내 폐업률 **72.4%** | 통계청 |
| 자금 병목 | 초기 창업자 투자유치 성공률 **3% 미만** | 한국벤처투자 |
| 정보 비대칭 | 지원사업 정보 획득 경로: 지인 추천 **58%** | 창업진흥원 실태조사 |

### 1.3 투자자 측 Pain Points

| 문제 | 현황 (2026) | 데이터 소스 |
|------|-------------|-------------|
| 딜소싱 비효율 | 초기 VC 심사 대비 투자 전환율 **2-3%** | KVCA |
| 정보 검증 비용 | 건당 실사(DD) 비용 **500만~2,000만원** | 업계 평균 |
| 정부지원 이력 불투명 | 창업자의 정부지원 수혜 이력 통합 조회 **불가** | — |
| 지역 딜 사각지대 | 수도권 외 투자 비중 **15% 미만** | 벤처투자 동향 |
| 초기 단계 접근 한계 | Pre-Seed/Seed 전문 플랫폼 **부재** | — |

### 1.4 정부 측 Pain Points

| 문제 | 현황 (2026) | 데이터 소스 |
|------|-------------|-------------|
| 성과 추적 한계 | 지원 후 3년 이상 추적률 **20% 미만** | 감사원 |
| 중복 수혜 | 동일 기업 다부처 중복 수혜 **탐지 어려움** | 중기부 |
| 민간 연계 부족 | 정부지원 → 민간투자 연계율 **8.3%** | 창업진흥원 |
| 예산 대비 가시성 | 연 **~9조원** 집행, 통합 성과 대시보드 **미비** | 기획재정부 |

## 2. 사업 목적: 삼각 플랫폼 (Triangle Platform)

### 2.1 비전

> **"정부지원과 민간투자를 연결하여, 대한민국 창업 생태계의 정보 비대칭을 해소하는 삼각 플랫폼"**

### 2.2 삼각 플랫폼 구조

```
                    ┌─────────────────┐
                    │    정부 (Gov)     │
                    │  Active Enabler  │
                    │                 │
                    │ • 예산 활용 최적화  │
                    │ • 성과 추적 대시보드 │
                    │ • 생태계 활성화 증명 │
                    └────┬───────┬────┘
                         │       │
              예산/인증    │       │   성과 데이터
              지원사업     │       │   투자 연계율
                         │       │
                    ▼    │       │    ▼
        ┌────────────┐  │       │  ┌────────────┐
        │ 창업자       │◄─┘       └─►│ 투자자       │
        │ (Founders)  │              │ (Investors) │
        │             │              │             │
        │ • 정보 + 도구 │◄────────────►│ • 딜소싱     │
        │ • 커뮤니티    │  G2E Score   │ • 검증       │
        │ • 정부지원    │  G-IR Profile│ • 연결       │
        │ • 투자유치    │  IR Room     │ • 포트폴리오   │
        └────────────┘              └────────────┘
```

### 2.3 각 축의 핵심 가치

**창업자 축 (Founders Axis)**

| 구성요소 | 설명 | 핵심 기능 |
|----------|------|-----------|
| **정보** | 정부지원 + 시장 인사이트 통합 | Gov-Support Radar, 산업 리포트 |
| **도구** | 사업 운영 실전 도구 | G-IR Profile 자동 생성, 사업계획서 템플릿 |
| **커뮤니티** | 동료 창업자 네트워크 | Fail Forward 밋업, 업종별 그룹 |

**투자자 축 (Investors Axis)**

| 구성요소 | 설명 | 핵심 기능 |
|----------|------|-----------|
| **딜소싱** | 정부 검증된 초기 딜 파이프라인 | G2E Score 기반 필터링 |
| **검증** | 정부지원 이력 + 실적 통합 검증 | G-IR Profile 열람, Due Diligence 보조 |
| **연결** | 창업자와 직접 소통 채널 | Anonymous IR Room, Reverse Pitch |

**정부 축 (Government Axis — Active Enabler)**

| 구성요소 | 설명 | 핵심 기능 |
|----------|------|-----------|
| **예산 활용** | 지원사업 매칭 최적화 | AI 기반 적합도 매칭 |
| **성과 추적** | 지원 후 성장 모니터링 | 성장 대시보드, 투자 연계율 |
| **생태계** | 민관 연계 활성화 증명 | 정부지원 → 민간투자 전환율 데이터 |

### 2.4 플라이휠 (Flywheel) 모델

```
    ┌──────────────────────────────────────────────────┐
    │                                                  │
    │   ① 정부 예산 투입 (연 ~9조원 지원사업)             │
    │         │                                        │
    │         ▼                                        │
    │   ② 창업자 검증 (정부 선발 = 1차 필터)              │
    │         │                                        │
    │         ▼                                        │
    │   ③ G2E Score 생성 (정부지원 이력 → 투자 신뢰 신호) │
    │         │                                        │
    │         ▼                                        │
    │   ④ 투자자 딜소싱 (검증된 파이프라인 접근)           │
    │         │                                        │
    │         ▼                                        │
    │   ⑤ 투자 집행 → 성장 (민간자본 유입)                │
    │         │                                        │
    │         ▼                                        │
    │   ⑥ 성과 증명 → 정부 (예산 ROI 가시화)              │
    │         │                                        │
    │         └──────────► ① 로 순환 (추가 예산 확보 근거) │
    │                                                  │
    └──────────────────────────────────────────────────┘
```

## 3. 타겟 고객 세분화

### 3.1 고객 구조: 2대 고객 + 1 Enabler

```
┌─────────────────────────────────────────┐
│          PRIMARY CUSTOMERS              │
│                                         │
│  ┌─────────────┐   ┌─────────────┐     │
│  │ Customer A   │   │ Customer B   │     │
│  │ 창업자        │   │ 투자자        │     │
│  │ (Freemium)   │   │ (SaaS/건당)  │     │
│  └─────────────┘   └─────────────┘     │
│                                         │
├─────────────────────────────────────────┤
│          ACTIVE ENABLER                 │
│                                         │
│  ┌─────────────────────────────────┐   │
│  │ 정부 / 공공기관 (B2G 계약)       │   │
│  └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 3.2 Customer A: 창업자 (Founders)

**세그먼트 A-1: 소상공인**

| 항목 | 상세 |
|------|------|
| **규모** | 약 690만 사업체 (2025 통계청) |
| **페르소나** | 김민수 (42세), 서울 마포구 카페 운영 3년차 |
| **핵심 니즈** | 정부지원금 정보, 세무/노무 실전 팁, 매출 증대 전략 |
| **Pain Point** | "어떤 지원사업이 나한테 맞는지 모르겠다" |
| **과금 의향** | 낮음 → **Freemium** (무료 정보 + 유료 매칭) |
| **플랫폼 가치** | Gov-Support Radar 맞춤 추천, 동종업계 커뮤니티 |

**세그먼트 A-2: 초기 스타트업**

| 항목 | 상세 |
|------|------|
| **규모** | 연 신규 법인 약 13만개 (2025) |
| **페르소나** | 이지영 (31세), AI 기반 물류 SaaS, 창업 1년차, TIPS 도전 중 |
| **핵심 니즈** | 투자유치, 정부 R&D 과제, IR 자료 작성, 멘토링 |
| **Pain Point** | "정부지원 받은 게 투자자한테 어떻게 보일지 모르겠다" |
| **과금 의향** | 중 → **Pro Plan** (G-IR Profile, 투자자 매칭) |
| **플랫폼 가치** | G2E Score 생성, G-IR Profile, Investor Hub 접근 |

**세그먼트 A-3: 예비 창업자**

| 항목 | 상세 |
|------|------|
| **규모** | 창업 희망자 약 80만명 (창업진흥원 2025 실태조사) |
| **페르소나** | 박준호 (27세), 대기업 퇴사 예정, F&B 테크 아이디어 보유 |
| **핵심 니즈** | 창업 로드맵, 시장 검증, 초기 자금 확보 방법 |
| **Pain Point** | "어디서부터 시작해야 할지 모르겠다" |
| **과금 의향** | 낮음 → **Free Tier** (전환 유도) |
| **플랫폼 가치** | 온보딩 퀴즈, 맞춤 로드맵, Fail Forward 콘텐츠 |

**세그먼트 A-4: 글로벌 진출 희망 창업자**

| 항목 | 상세 |
|------|------|
| **규모** | CES 2025 한국 참가 1,031개사 + K-Startup Grand Challenge 등 |
| **페르소나** | 최서연 (35세), 바이오테크 스타트업, Series A, 미국 시장 진출 준비 |
| **핵심 니즈** | Delaware Flip 가이드, 글로벌 VC 연결, 해외 전시회 정보 |

## 7. Fail Forward 전략

### 7.1 철학: 실패를 자산으로 전환

> "실패는 숨겨야 할 치부가 아니라, 생태계 전체가 학습해야 할 가장 값비싼 데이터다."

한국 창업 생태계는 실패에 대한 **낙인(stigma)**이 강하다. 그러나 글로벌 관점에서 실패 데이터는 **희소하고 가치 있는 자산**이다.

| 글로벌 벤치마크 | 모델 | 규모 | 핵심 인사이트 |
|----------------|------|------|-------------|
| **FuckUp Nights** | 실패 스토리텔링 이벤트 | 330+ 도시 | 실패 공유가 커뮤니티를 만든다 |
| **FailCon** | 실패 분석 컨퍼런스 | Silicon Valley 기원 | 체계적 실패 분석이 성공률을 높인다 |
| **Failory** | Post-mortem 데이터베이스 | 300+ 사례 분석 | 실패 패턴은 반복되며 예방 가능하다 |
| **한국 실패박람회** | 정부 주도 실패 수용 | 연 1회 | 정부도 실패 수용 문화를 원한다 |
