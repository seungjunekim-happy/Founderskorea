## 📐 글 작성 표준 (모든 글 공통)

### 1. 프론트매터 형식
```yaml
---
title: "..."
subtitle: "..."
category: "실전노하우"   # 카테고리 신설 전까지 임시
date: "YYYY-MM-DD"
tags: ["사업자등록", "행정가이드", "예비창업자", ...]
cluster: "business-registration"   # 신규 필드, 클러스터 추적용
hub: false                          # 허브 글이면 true
---
```

### 2. 글 구조 템플릿
```
# H1 제목

> 한 줄 부제 (검색결과 메타디스크립션용)

## 도입 (200~300자)
- 독자 페인 포인트 한 문장
- 이 글 다 읽으면 무엇을 알 수 있는지

## 본문 (3~6개 H2 섹션)
- 각 섹션마다 H3 1~3개
- 표 1개 이상 (비교/체크리스트/요약)
- 외부 링크: 정부 공식 사이트만 (홈택스, 국세청, K-Startup)

## 자주 묻는 질문 (FAQ 3~5개)
- 구글 People Also Ask 노출용

## 다음 단계 (CTA + 관련 글 링크)
- 클러스터 내 다른 글 3~5개 링크
- 정부지원 매칭 도구 CTA 1개

---
**핵심 메시지**: (한 줄)
```

### 3. SEO 체크리스트
- [ ] 핵심 키워드 H1·H2·첫 문단·메타에 포함
- [ ] 글 길이 2,000자 이상 (허브 글은 3,500자+)
- [ ] 표 1개 이상
- [ ] 내부링크 3~5개 (다른 클러스터 글 또는 도구)
- [ ] 외부링크는 정부·공식 출처만
- [ ] FAQ 섹션 (스키마용)
- [ ] 이미지 alt 텍스트 포함 (이미지 추가 시)

## 🔗 클러스터 내부링크 구조

```
[허브] #1 개인사업자 등록 완전 가이드
   ├─ #2 개인 vs 법인 비교
   ├─ #3 홈택스 5분 신청
   ├─ #4 업종코드 100선
   ├─ #5 간이/일반과세
   ├─ #7 부업·N잡러
   ├─ #8 크리에이터
   ├─ #9 스마트스토어
   ├─ #12 사업장 주소
   └─ #13 등록 후 30일
```

**규칙:**
- 모든 위성 글은 허브 글로 1회 이상 링크
- 허브 글은 모든 위성 글로 링크 (목차 형태)
- 위성 글끼리도 관련성 있으면 상호 링크

## 🛠 기술/운영 사전 작업 (1주차 시작 전)

| 항목 | 우선순위 | 비고 |
|---|---|---|
| 새 카테고리 "행정가이드" 또는 "창업준비" 추가 | P1 | 일단 "실전노하우"로 임시 운영 가능. 5편 발행 후 신설 권장 |
| 태그 시스템 점검 — `사업자등록`, `세무`, `행정` 태그 통일 | P0 | 첫 글 쓸 때부터 정리 |
| 각 글에 `cluster` 필드 추가 가능한지 확인 | P2 | 추후 클러스터별 페이지 만들 때 필요 |
| 허브 글 sitemap 우선순위 0.9로 조정 | P2 | SEO 최적화 |
| Google Search Console 등록·확인 | P0 | 트래픽 측정 필수 |
